# 宿題管理アプリ

これは宿題管理アプリです。宿題をすぐに登録できるように、ボタン操作のみで
宿題を登録することができます。自分用に開発したアプリなので、デザインが初期のものだったり、宿題の科目の編集機能が無かったりします。

# 使用技術
Android Studio Arctic Fox | 2020.3.1 Patch4  
Room ver2.4.1  
Git  ver2.33.0.windows.2  
Github
